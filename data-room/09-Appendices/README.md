## Glossary

| Term | Definition |
|------|-----------|
| **Affiliate Marketing** | Performance-based marketing where publishers (affiliates) earn commissions for driving conversions (sales, leads, sign-ups) to brands |
| **CPA** | Cost Per Acquisition — fixed fee paid per conversion |
| **CPS** | Cost Per Sale — commission as percentage of sale value |
| **CPL** | Cost Per Lead — fee per qualified lead generated |
| **CVR** | Conversion Rate — percentage of clicks that result in a conversion |
| **ROAS** | Return on Ad Spend — revenue generated per dollar spent |
| **GMV** | Gross Merchandise Value — total value of goods/services sold |
| **Publisher** | An individual or company that promotes brand offers through content, coupons, social media, or other digital channels |
| **Advertiser / Brand** | A company that pays for affiliate-driven conversions |
| **Attribution** | The process of determining which publisher/click/impression deserves credit for a conversion |
| **Server-to-Server (S2S) Postback** | A tracking method where conversion data is sent directly between servers rather than through browser cookies |
| **Click Farm** | Fraudulent operation using human or automated agents to generate fake clicks |
| **Cookie Stuffing** | Fraudulent technique where affiliate cookies are placed on users' browsers without their knowledge |
| **VPN Masking** | Use of VPNs to disguise traffic origin — used fraudulently to make traffic appear from target geographies |
| **Take Rate** | Platform's commission as percentage of total transaction value |
| **NRR** | Net Revenue Retention — percentage of revenue retained from existing clients year-over-year, including expansion |
| **ARPC** | Average Revenue Per Client |
| **CAC** | Customer Acquisition Cost |
| **LTV** | Lifetime Value of a customer |
| **MENA** | Middle East and North Africa |
| **GCC** | Gulf Cooperation Council — UAE, KSA, Bahrain, Kuwait, Oman, Qatar |
| **TAM** | Total Addressable Market |
| **SAM** | Serviceable Addressable Market |
| **SOM** | Serviceable Obtainable Market |
